# Local Time Google Calendar
This website can generate a link of a google calendar that always displays in the viewer's local time. This was made for the purpose of online events.

Just copy and paste the `src` attribute from the embed link after a `?` after the url like this:
- `?src=your_calendar@group.calendar.google.com`

So the final link would be like this:
- `https://erallie.github.io/Local-Time-Google-Calendar/?src=your_calendar@group.calendar.google.com`

You can find the website at https://erallie.github.io/Local-Time-Google-Calendar/
